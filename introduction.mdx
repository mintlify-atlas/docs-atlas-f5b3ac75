## Overview

Hotel Reservation System is a comprehensive SaaS application designed to streamline hotel operations. Built with Express.js and MySQL, it provides a complete suite of tools for managing your hotel's daily operations, from client registration to room inventory and reservation management.

## Technology Stack

The system is built on a modern, reliable technology stack:

- **Backend**: Express.js - Fast, unopinionated web framework for Node.js
- **Database**: MySQL - Robust relational database for data persistence
- **Frontend**: Vanilla JavaScript with Bootstrap - Clean, responsive user interface
- **Architecture**: MVC pattern with separate controllers, routers, and models
